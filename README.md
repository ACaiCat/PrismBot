## 关于本项目

这是一个基于 [Sora](https://github.com/Hoshikawa-Kaguya/Sora) 框架的 Terraria [TShock](https://github.com/Pryaxis/TShock) 插件式 QQ 机器人

这可能是 TShock QQ 机器人的终极解决方案

准确的来说，这更像一个插件框架

你可以使用 C# 编写插件来实现任何你想实现的功能
